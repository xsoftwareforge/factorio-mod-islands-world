# Islands World (Factorio 2.0)

Map generator preset "Many islands" that creates many small islands.
The mod also ensures iron, copper, stone, and coal exist inside the starting area.

## Installation
1. Copy this folder to your Factorio mods directory.
2. Enable the mod in-game.

## Usage
- New game: choose the map-gen preset "Many islands" (id: `kap_islands`).
- Existing save: run `/enable_islands_world` once to switch elevation and place missing starting resources.
- The mod will retry placement while chunks generate until all four resources exist.

## Compatibility
- Factorio 2.0
- Locales: en, ja
